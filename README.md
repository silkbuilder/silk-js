# SilkBuilder JavaScript Library

A series of JavaScript classes and functions used by the SilkBuilder Framework. These provide support for custom tags interaction and programming utility tools.

The library is included in the SilkBuilder-core and SilkClient-core packages as a compiled and minified file called `silkObjects.min.js`.

The documentation for this library can be found [here](https://docs.silkbuilder.com/js_object).

Learn about SilkBuilder at [https://silkbuilder.com](https://silkbuilder.com)
